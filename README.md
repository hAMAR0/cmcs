# cmcs - somewhat trivial C minecraft server

A personal project for learning server architecture (non-blocking sockets, leaving the fork()/threads model) and reverse engineering minecraft server protocol.

## Building

`make`

## Useful information:
+ Minecraft protocol FAQ - https://minecraft.wiki/w/Java_Edition_protocol/FAQ
+ Blocking vs. non-blocking sockets - https://www.scottklement.com/rpg/socktut/nonblocking.html
